# Bubble Sort implemented with Ruby
Build a method #bubble_sort that takes an array and returns a sorted array. It must use the bubble sort methodology (using #sort would be pretty pointless, wouldnâ€™t it?).

## Instructions
1. Download and Install the latest [Ruby](https://www.ruby-lang.org)
2. Run `irb -r ./bubble_sort.rb`
3. Run:
```ruby
$ bubble_sort([4,3,78,2,0,2])
```
Expected output: 
```ruby
$ [0,2,2,3,4,78]
```
